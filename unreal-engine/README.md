# Unreal Engine 5.6

이 디렉토리에 UE 5.6을 설치해야 합니다.

## 다운로드

- 공식 문서: https://dev.epicgames.com/documentation/ko-kr/unreal-engine/linux-development-quickstart-for-unreal-engine?application_version=5.6
- GitHub (Epic Games 계정 필요): https://github.com/EpicGames/UnrealEngine/tree/5.6

## 설치 후 확인

```bash
ls Engine/Binaries/Linux/UnrealEditor
```

## 참고

- `Engine/Binaries/Linux/UnrealEditor` 실행 시 언리얼 엔진 에디터 실행됨
- Colosseum AirSim 플러그인과 함께 사용
